# Hopla - L'app de courses entre voisins à Strasbourg

Une application ultra-simple pour faire ses courses ou gagner de l'argent en livrant.

## Fonctionnalités

- **Simple et Rapide** : Pas d'inscription complexe. Commencez immédiatement.
- **2 Modes** :
  - **Je veux des courses** : Créez votre liste en quelques clics.
  - **Je veux livrer** : Consultez les missions disponibles et acceptez-les.
- **Suivi** : Suivez vos missions en cours.

## Comment lancer l'application

1. Ouvrez un terminal dans ce dossier.
2. Installez les dépendances (déjà fait normalement) :
   ```bash
   npm install
   ```
3. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```
4. Ouvrez `http://localhost:3000` dans votre navigateur.

## Tech Stack

- **Framework** : Next.js 14 (App Router)
- **Langage** : TypeScript
- **Style** : CSS Vanilla (Design Premium)
- **Icônes** : Lucide React
